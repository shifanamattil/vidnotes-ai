# VidNotes AI

AI-powered YouTube video summarizer that converts video transcripts into structured notes using HuggingFace Transformers.

## Features

- AI-generated video notes
- YouTube transcript extraction
- React frontend
- Flask backend
- HuggingFace NLP summarization
- Responsive modern UI

## Tech Stack

Frontend:
- React
- Vite
- CSS

Backend:
- Flask
- Transformers
- PyTorch

AI:
- FLAN-T5
